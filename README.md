# javascript-o-inicio
aprendizado inicial em javascript

## Fundamentos Básicos

-Sintaxe básica (variáveis, operadores, comentários)

-Tipos de dados (string, number, boolean, null, undefined, object)

-Declaração de variáveis (var, let, const)

-Estruturas de controle (if, else, switch)

-Laços (for, while, do...while)

-Funções (declaração, expressão, arrow functions)

-Escopo (global, local, de bloco)

## Trabalhando com Dados

-Arrays (criação, métodos básicos como push, pop, length)

-Objetos (criação, propriedades, métodos)

-Manipulação básica de strings e números

-Manipulação do DOM (Document Object Model)

-Selecionar elementos (getElementById, querySelector)

## Alterar conteúdo e estilos de elementos

-Eventos básicos (click, input, submit)

-Manipular classes CSS

-Conceitos Importantes

-Tipagem dinâmica e coerção de tipos

-Funções de callback

-Promises (conceito básico)

-JSON (leitura e escrita)

## Ferramentas e Ambiente

-Console do navegador (console.log, debugger)

-Uso básico do navegador para rodar e testar código JS

-Noções básicas de HTML e CSS para interação com JavaScript
